# 🔍 Project Overview

This project analyzes Zomato food delivery data to uncover insights into customer behavior, restaurant performance, and sales trends. Using SQL, Excel (Power Query), and Power BI, raw data is transformed into meaningful business insights.

# 🎯 Objectives
- Analyze food order patterns and customer preferences
- Identify peak order hours and high-demand locations
- Evaluate restaurant performance and sales trends
- Improve decision-making using data-driven insights

# 🛠️ Tools & Technologies
- SQL – Data extraction & analysis (40%)
- Excel (Power Query) – Data cleaning & transformation (30%)
- Power BI – Dashboard & visualization (30%)

# 📊 Data Preprocessing
- Cleaned and structured large-scale Zomato dataset
- Handled missing values and removed duplicates
- Standardized columns (city, cuisine, ratings)
- Transformed raw data using Power Query
- Improved data processing efficiency by 50%

# 🧠 SQL Analysis
- Used advanced SQL techniques:
- JOINs for multi-table analysis
- GROUP BY and aggregations
- Subqueries for deeper insights
- Generated insights on:
- Top-performing restaurants
- Customer ordering behavior
- Revenue trends by location
  
# 📈 Exploratory Data Analysis (EDA)
- Identified:
  - Peak ordering hours
  - Popular cuisines
  - High-demand locations
  - Analyzed customer preferences and ordering frequency
  - Discovered trends in ratings vs order volume

# 📊 Dashboard Features (Power BI)
- Sales Performance Overview
- Restaurant Ratings & Reviews Analysis
- Customer Order Trends
- Peak Hours & Location Insights
- Interactive filters and slicers
